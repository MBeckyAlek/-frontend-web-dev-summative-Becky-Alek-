About Student Finance Tracker

This is a Student Finance Tracker application developed for educational purposes. It provides a comprehensive tool for managing student budgets, tracking expenses, and analyzing spending patterns.

Key Features

  • Transaction Management: Add, edit, and delete financial transactions
  • Dashboard Analytics: View spending statistics and trends
  • Category Organization: Organize expenses by categories
  • Advanced Search: Regex-based search with highlighting
  • Data Persistence: Auto-save with JSON import/export
  • Responsive Design: Works on all devices
  • Accessibility: Full keyboard navigation and screen reader support
